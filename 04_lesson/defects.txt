### Баг-репорт №1: Ошибка в методе contains() при передаче пустой строки

**Шаги для воспроизведения:**
1. Вызвать метод `contains("SkyPro", "")`.

**Ожидаемый результат:**
Метод возвращает `False`, так как пустая строка не должна считаться частью строки.

**Фактический результат:**
Метод возвращает `True`, так как пустая строка считается присутствующей в строке.

**Приоритет:**
Средний (Medium) 


**Причина:**
Встроенные методы Python считают пустую строку всегда присутствующей.

**Решение:**
Добавить проверку на пустую строку в метод `contains()`:
```python
def contains(self, string: str, symbol: str) -> bool:
    if symbol == "":
        return False
    return symbol in string


### Баг-репорт №2: Ошибка в методе starts_with() при передаче пустой строки

**Шаги для воспроизведения:**
Вызвать метод starts_with("SkyPro", "").

**Ожидаемый результат:**
Метод возвращает False, так как пустая строка не должна считаться началом строки.

**Фактический результат:**
Метод возвращает True, так как пустая строка считается началом любой строки.

**Приоритет:**
Низкий (Low)

**Причина:**
Встроенные методы Python считают, что любая строка начинается с пустой строки.

**Решение:**
Добавить проверку на пустую строку в метод starts_with():

def starts_with(self, string: str, symbol: str) -> bool:
    if symbol == "":
        return False
    return string.startswith(symbol)


### Баг-репорт №3: Ошибка в методе end_with при передаче пустой строки

**Шаги для воспроизведения:**
Вызвать метод end_with("SkyPro", "").

**Ожидаемый результат:**
Метод возвращает False, так как пустая строка не должна считаться началом строки.

**Приоритет:**
Низкий (Low)

**Фактический результат:**
Метод возвращает True, так как пустая строка считается началом любой строки.

**Причина:**
Встроенные методы Python считают, что любая строка начинается с пустой строки.

**Решение:**
Добавить проверку на пустую строку в метод end_with():

def end_with(self, string: str, symbol: str) -> bool:
    if symbol == "":
        return False
    return string.endswith(symbol)
